Mi experiencia

Soy un entusiasta del desarrollo web front-end con un enfoque en la creación de interfaces atractivas y funcionales. Aunque no tengo experiencia laboral formal, he adquirido un sólido conjunto de habilidades en HTML, CSS, Sass y Bootstrap a través de proyectos personales y estudios autodidactas. Actualmente, me encuentro enfocado en ampliar mis conocimientos y dominar JavaScript para agregar interactividad a mis creaciones web.

Conocimientos Técnicos

HTML

Poseo un amplio dominio de HTML y puedo crear estructuras de página bien organizadas y semánticamente correctas. Estoy familiarizado con las últimas especificaciones y buenas prácticas de accesibilidad.

CSS

Tengo experiencia en el uso de CSS para dar estilo a sitios web, aplicando diseños atractivos y responsivos. Puedo utilizar selectores, pseudo-clases y propiedades avanzadas para lograr el aspecto deseado.

Sass

He trabajado con Sass, un preprocesador CSS, para aprovechar características como variables, anidamiento y mixins. Esto me permite escribir estilos más limpios y eficientes, ahorrando tiempo y facilitando el mantenimiento.

Bootstrap

Estoy familiarizado con el framework Bootstrap y puedo utilizar su sistema de rejilla y componentes para desarrollar interfaces rápidas y receptivas. Comprendo cómo personalizar y adaptar Bootstrap para cumplir con los requisitos específicos de un proyecto.

JavaScript

Aunque actualmente me encuentro en la etapa de estudio de JavaScript, ya tengo conocimientos básicos y entiendo los conceptos fundamentales, como variables, funciones y estructuras de control. Estoy trabajando en la creación de pequeñas aplicaciones y proyectos para fortalecer mis habilidades.
